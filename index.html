Venti

How to Play:

Match numbered tiles to form higher numbers. Reach 20 to win.

Tiles drop down to lowest unoccupied positions.

Every few seconds, more tiles will be added to the bottom.

The game is over if the tiles reach the top.

About:

Venti, inspired by the puzzle game Twenty, was built using JavaScript to handle core game logic and jQuery to manage user interface. Dragging functionality was implemented through jQuery's UI library, enabling the user to manually detach an element from the DOM and append it elsewhere. With every move, the game determines final positions of affected tiles and reattaches them accordingly.
